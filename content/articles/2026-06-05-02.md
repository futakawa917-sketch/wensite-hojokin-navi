## 「自分は補助金をもらえるの？」その疑問、この記事で全部解決します

補助金の活用を検討しているとき、多くの方が最初にぶつかる壁が「そもそも自分は対象になるのか」という疑問です。個人事業主なのか、株式会社なのか、NPO法人なのか——同じ「事業者」でも、補助金ごとに申請できる主体は異なります。条件を確認せずに申請書を仕上げてしまい、受付段階で門前払いになるケースは、補助金の現場では決して珍しくありません。

この記事では、**個人・法人・NPO**という3つの申請主体ごとに、補助金の対象要件をわかりやすく整理します。さらに「対象外になりやすいケース」や「申請前に自分でできる要件チェック」まで網羅しているので、読み終わるころには迷いなく自身の立ち位置を把握できるはずです。

## 補助金における「対象者」とは何か？基本概念を整理する

補助金の「対象者（申請主体）」とは、補助金を申請・受給できる資格を持つ主体のことです。一般的な補助金では、以下のような要件が設けられています。

- 事業を営んでいること（事業実施主体であること）
- 一定の規模要件を満たすこと（中小企業・小規模事業者など）
- 法的に適切な形態であること（登記・開業届等）
- 税金の滞納がないこと
- 反社会的勢力と関係がないこと

2026年時点で国や自治体が運営する主要な補助金の多くは、「中小企業・小規模事業者」を対象に設計されています。中小企業基本法（昭和38年法律第154号）に基づく定義では、製造業の場合は**資本金3億円以下または従業員300人以下**、卸売業では**資本金1億円以下または従業員100人以下**が中小企業の目安です。

ただし、この「対象者」の定義は補助金ごとに異なるため、「中小企業なら何でも申請できる」という認識は危険です。補助金の公募要領を必ず確認することが前提となります。

## 個人事業主が補助金を申請するときの条件と注意点

### 個人事業主が申請できる主な補助金

個人事業主とは、法人格を持たずに事業を営む人のことです。税務署への開業届の提出が事業実施の証明として機能します。2026年時点で個人事業主が申請できる代表的な補助金には以下があります。

| 補助金名 | 補助上限額 | 個人事業主の可否 |
|---|---|---|
| 小規模事業者持続化補助金 | **最大250万円** | ○（開業届必須） |
| IT導入補助金 | **最大450万円** | ○（gBizID必要） |
| ものづくり補助金 | **最大1,250万円** | ○（一部） |
| 事業再構築補助金 | **最大7,000万円** | ○（売上要件あり） |
| 創業補助金（地方版） | **最大200万円** | ○（開業前後） |

個人事業主が申請するうえで特に注意すべき点は以下の3つです。

**① 開業届の提出が前提**
多くの補助金では、税務署への「個人事業の開業・廃業等届出書」の提出が必須要件です。副業や兼業で収入を得ているだけでは対象外になる可能性があります。

**② 「業種」が制限される場合がある**
補助金によっては、特定の業種を除外しているものがあります。例えば、風俗営業法に基づく事業者は多くの補助金で対象外です。農業・林業・漁業なども別途補助制度が設けられているため、一般的な中小企業向け補助金の対象外になることがあります。

**③ 確定申告の実績が必要な場合も**
事業再構築補助金のように、売上の減少率を証明するために過去の確定申告書の提出を求めるケースがあります。開業後間もない事業者は、申告実績がないため対象外になることがあります。

> **Aさん（42歳・フリーランスのWebデザイナー・東京都在住）のケース**
> 
> Aさんは、クライアントが増えてきたのを機に、IT導入補助金を使って業務管理システムを導入しようと考えました。ところが申請直前に、開業届を提出していないことに気づきます。Aさんは個人事業としてではなく「副業」として活動しており、確定申告も雑所得で処理していたのです。結果として、その年度の申請は断念せざるを得ませんでした。翌年、正式に開業届を提出し、青色申告に切り替えてから改めて申請し、**約150万円のシステム導入費用の補助**を受けることに成功しました。
> 
> 「開業届を出すだけでこんなに選択肢が広がるとは思っていませんでした」とAさんは振り返ります。

## 法人が補助金を申請するときの条件と主体の多様性

### 「法人」は一枚岩ではない

一口に「法人」と言っても、その種類は多様です。補助金の申請において特に重要なのは、以下の区別です。

| 法人種別 | 特徴 | 補助金申請の主な注意点 |
|---|---|---|
| 株式会社・合同会社 | 営利目的の一般的な法人 | 規模要件・業種制限の確認 |
| 一般社団法人 | 非営利・営利どちらもあり | 「非営利型」か否かで税制が変わる |
| 一般財団法人 | 財産を持つ非営利法人 | 事業内容の公益性が求められることも |
| 医療法人 | 病院・クリニック等 | 補助金の種類が限定的 |
| 学校法人 | 教育機関 | 文部科学省系の補助金が中心 |
| 農事組合法人 | 農業経営 | 農水省系の補助金が対象 |

株式会社や合同会社などの「営利法人」が中小企業向け補助金を申請する場合、資本金・従業員数・直近の決算書などの確認が必要です。

### 法人設立直後の申請に潜む落とし穴

設立して**1期目の法人**は、決算書の提出を求められる補助金の多くで申請が難しくなります。例えば事業再構築補助金では、コロナ前後の売上比較を求めますが、設立後間もない法人は比較対象の売上データ自体が存在しません。

一方、**設立間もない法人こそ有利な補助金**も存在します。創業補助金や自治体の起業支援制度は、設立から**3年以内**を対象条件としているケースが多く、むしろ積極的に活用すべきタイミングです。

## NPO法人が補助金を活用するための要件と特有の難しさ

![イメージ画像](https://images.unsplash.com/photo-1560448204-e02f11c3d0e2?w=800&q=80)

### NPO法人は補助金を申請できるのか

NPO法人（特定非営利活動法人）は、特定非営利活動促進法（平成10年法律第7号）に基づいて設立された非営利法人です。2026年時点でも補助金申請の対象となるケースは多く、特に社会福祉・子育て・環境・地域活性化分野ではNPOを積極的に対象に含む補助金が増えています。

しかし、NPO法人特有の注意点があります。

**① 「事業収益」の概念が一般企業と異なる**
NPO法人は会費・寄付・助成金・事業収益の複合で運営されています。補助金によっては「営利事業を行う主体」を対象にしているため、事業の位置づけを明確にする必要があります。

**② 活動分野が制限される場合がある**
小規模事業者持続化補助金などは、NPO法人でも申請できますが、「特定非営利活動に係る事業」として認定されている必要があります。

**③ 財務諸表の整備が弱い団体は不利**
NPO法人の中には、会計体制が不十分な団体も少なくありません。補助金では通常、財務諸表（活動計算書・貸借対照表）の提出が求められます。

### NPOが積極活用できる補助金・助成金の例

| 制度名 | 対象 | 上限額 |
|---|---|---|
| 地域づくり推進事業補助金 | NPO・自治体等 | **最大500万円** |
| 共助社会づくり推進事業 | NPO法人 | **最大300万円** |
| 社会福祉振興助成事業 | NPO・社会福祉法人 | **最大1,000万円** |
| 地域創生推進交付金 | NPO含む地域団体 | 事業規模による |

NPO法人が補助金を活用する場合、国の補助金だけでなく、地方自治体や民間財団の「助成金」も視野に入れることをお勧めします。助成金は一般的に審査基準が明確で、社会貢献性を重視するNPOの活動と親和性が高い傾向があります。

> **Bさん（50代・NPO法人代表・大阪府在住）のケース**
> 
> 放課後の子ども支援を行うNPO法人を運営するBさんは、施設の老朽化した設備を刷新するため、補助金の活用を検討しました。最初は「NPOは対象外では？」と思い込んでいたため、申請を躊躇していたといいます。しかし地元の商工会議所に相談したところ、小規模事業者持続化補助金の対象であることを確認。さらに自治体の「子ども・子育て支援事業」補助金との併用が可能であることもわかりました。最終的に**合計で約280万円**の補助を受け、施設のバリアフリー化と学習用タブレット導入を実現しました。
> 
> 「NPOだからと最初から諦めていたら何も変わっていなかった」とBさんは話します。

## 申請前に確認すべき「対象者要件」チェックリスト

補助金の対象者かどうかを判断するために、申請前に以下のチェックリストで確認してみてください。

### ✅ 共通確認事項（全主体共通）

- [ ] 事業実態があるか（開業届・登記・定款等）
- [ ] 国税・地方税に滞納がないか
- [ ] 反社会的勢力との関係がないか
- [ ] 申請する事業が補助対象となる業種か
- [ ] 公募要領の「対象者」欄を最新版で確認したか

### ✅ 個人事業主固有の確認事項

- [ ] 税務署への開業届を提出しているか
- [ ] 確定申告（青色申告・白色申告）の実績があるか
- [ ] 従業員数・所得の規模が要件を満たしているか

### ✅ 法人固有の確認事項

- [ ] 法人登記が完了しているか
- [ ] 資本金・従業員数が中小企業要件を満たしているか
- [ ] 決算書（直近1〜2期分）を準備できるか
- [ ] 大企業の子会社・関連会社ではないか（みなし大企業の排除）

### ✅ NPO法人固有の確認事項

- [ ] 所轄庁（都道府県または内閣府）の認証を受けているか
- [ ] 特定非営利活動に関する事業として申請内容が整合しているか
- [ ] 活動計算書・貸借対照表を提出できる状態か
- [ ] 「非営利型」の一般社団法人と混同していないか

## 「みなし大企業」に注意——知らずに対象外になるケース

法人申請で特に見落とされがちなのが「**みなし大企業**」の問題です。

中小企業向けの補助金では、たとえ資本金が基準以下であっても、大企業が**一定割合以上の株式を保有**している場合は「みなし大企業」とされ、申請対象外になります。

2026年時点における代表的な「みなし大企業」の判定基準（補助金によって異なる）：

- 大企業が発行済株式総数の**1/2以上**を保有している場合
- 大企業が発行済株式総数の**2/3以上**を複数の大企業が保有している場合
- 役員・従業員の過半数が大企業の出向者である場合

フランチャイズ加盟店や親会社を持つ中小企業は、この要件を事前に確認することが不可欠です。[補助金申請で不採択になる7つの理由と採択率を上げる具体的対策](/articles/2026-04-16-01)でも解説していますが、対象者要件の確認漏れは最も初歩的かつ致命的な失敗です。

│

│    ├─ 設備投資・生産性向上 → ものづくり補助金

## 対象者要件を満たしていても「落とし穴」がある3つの状況

要件を満たしているにもかかわらず、実質的に申請が難しくなるケースが3つあります。

### ① 同一事業への二重申請

同じ事業・設備投資に対して、複数の補助金を重複して申請することは原則禁止されています。ただし、補助率や補助対象経費が異なれば**別の経費で申請できる**場合もあります。この点は公募要領の「他の補助金との併用」欄で確認が必要です。

### ② 過去の採択・受給歴による制限

補助金によっては、過去に同種の補助金を受給してから**一定期間内の再申請を認めない**ものがあります。小規模事業者持続化補助金では、直近の補助事業が完了してから原則として次の申請が可能になります。

### ③ 申請時点の事業実態

申請時点で「これから始める予定」の事業に対して補助金を申請できるかどうかは、制度によって大きく異なります。創業補助金のように事業開始前でも申請できるものもあれば、すでに事業を営んでいることが前提の補助金もあります。

「申請できる状態か」は、[補助金の採択率を上げる方法｜プロが教える5つの戦略](/articles/2026-04-16-03)と合わせて事前に整理しておくと、戦略的な申請計画が立てやすくなります。

## まとめ

![イメージ画像](https://images.unsplash.com/photo-1582407947092-5e21e1f8b40a?w=800&q=80)

補助金の「対象者」は、制度ごとに細かく定められています。本記事のポイントを整理します。

- **個人事業主**は開業届の提出・確定申告実績が申請の前提になる
- **法人**は「みなし大企業」の判定に注意。設立直後は使える制度を選ぶ
- **NPO法人**は対象外と思い込まずに、まず公募要領を確認する
- **全主体共通**で、税金の滞納・反社要件・業種制限を事前に確認する
- 申請前の要件チェックリストを使って「土俵に立てるか」を先に確認する
- 対象者であっても、二重申請禁止・再申請制限・事業実態要件に注意が必要

「自分は対象外だろう」という思い込みで補助金の活用をあきらめてしまうのは非常にもったいないことです。一方で、「きっと対象のはず」という思い込みのまま申請書類の作成を進めるのも時間とコストの無駄になります。まずは冷静に要件を確認することから始めてください。
