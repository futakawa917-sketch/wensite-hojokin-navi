## 採択通知が届いた後こそが本番――「受け取れなかった」は実は珍しくない

補助金の採択通知が届いたとき、「これで一安心」と感じる事業者の方は多いでしょう。しかし現場では、採択後の手続きでつまずき、**補助金を受け取れないまま終わるケース**が毎年一定数発生しています。

交付申請の遅延、実績報告の書類不備、期限切れによる失効――こうした「採択後の失敗」は、申請書作成とは別のスキルが必要です。

この記事では、補助金の交付申請から実績報告・確定検査・入金に至るまでの**全ステップを具体的な手順と注意点**とともに解説します。2026年時点の最新制度に基づいた情報をお届けしますので、採択済みの方はもちろん、これから申請を検討している方にもぜひ読んでいただきたい内容です。

## 補助金の「採択」はゴールではなくスタートライン

補助金の仕組みを正確に理解するところから始めましょう。多くの補助金は「**後払い方式（後払いの補助金）**」を採用しています。つまり、事業者がまず自己負担で事業を実施し、その後に審査を経て補助金が振り込まれる仕組みです。

一般的な流れは以下のとおりです。

```
公募・申請 → 採択通知 → 交付申請 → 交付決定 
→ 事業実施（発注・契約・支払い） → 実績報告 
→ 確定検査 → 補助金額の確定 → 入金（補助金振込）
```

採択はあくまでも「この事業に補助金を出す可能性がある」という段階です。**実際に補助金が振り込まれるのは、事業を実施した後の審査を経てから**になります。

### 後払い方式が生む「資金繰りリスク」

後払い方式の場合、事業者は先に全額を自己資金で立て替える必要があります。たとえば補助上限額が**500万円**の補助金で、事業費総額が**800万円**だった場合、まず800万円を支払い、後から**補助率に応じた金額**が戻ってくるイメージです。

資金調達が難しい場合は、補助金つなぎ融資（日本政策金融公庫など）を活用する方法もあります。事業計画を立てる段階から資金繰りを考えておくことが重要です。

## STEP1｜交付申請：採択通知を受け取ったら最初にやること

採択通知を受け取った後、最初に行うのが「**交付申請**」です。交付申請とは、採択された計画に基づいて「補助金の交付を正式に申請する」手続きのことです。

交付申請をしなければ、補助金を受け取る権利が発生しません。また、**交付決定が出る前に事業（発注・契約）を開始してしまうと、補助対象外になる**ケースが多いため注意が必要です。

### 交付申請の主な提出書類

| 書類の種類 | 内容 | 注意点 |
|---|---|---|
| 交付申請書 | 所定様式・事業概要 | 採択内容と齟齬がないか確認 |
| 経費明細書（見積書添付） | 補助対象経費の内訳 | 相見積もりが必要な場合あり |
| 事業実施計画書 | スケジュール・実施体制 | 採択申請書との整合性 |
| 銀行口座情報 | 振込先口座 | 法人名義であること |
| 登記簿謄本・決算書 | 直近のもの | 発行から3ヶ月以内が多い |

> **ポイント**: 相見積もりが必要な金額基準は補助金によって異なります。たとえば小規模事業者持続化補助金では**50万円以上（税抜）の場合に2社以上の相見積もり**が求められるケースが一般的です。基準を確認してから発注先を絞り込むようにしてください。

## STEP2｜交付決定：事業開始は必ずこの通知後に

交付申請を提出すると、事務局から審査が行われ「**交付決定通知書**」が届きます。この通知書が届いた日付が、補助事業の「開始日」とみなされます。

**交付決定前に契約・発注・支払いを行った経費は、補助対象から除外されます。**これは非常に重要なルールです。

2026年時点において、各補助金の交付決定までの期間の目安は以下のとおりです。

| 補助金名 | 交付申請から交付決定までの目安 |
|---|---|
| ものづくり補助金 | **3〜6週間**程度 |
| 小規模事業者持続化補助金 | **4〜8週間**程度 |
| IT導入補助金 | **1〜2週間**程度（ITツール登録後） |
| 事業再構築補助金 | **4〜6週間**程度 |

「交付決定が遅くて補助事業期間内に終わらない」という焦りから、決定前に発注してしまうケースが後を絶ちません。スケジュールは余裕を持って設計してください。

## STEP3｜事業実施：証憑書類を「その都度」揃える習慣が命綱

交付決定後、いよいよ事業を実施します。この段階で**最も重要なのは、証憑書類（しょうひょしょるい）を都度・正確に保管すること**です。

証憑書類とは、経費の支払い事実を証明する書類のことです。後から「払った事実はあるが書類がない」という状況になると、実績報告で大きな問題になります。

### 経費区分別の必要書類チェックリスト

**【機械装置・システム構築費】**
- ☑ 見積書（採択前から持っている場合でも改めて取得）
- ☑ 発注書・注文書（日付・金額・仕様が明記されたもの）
- ☑ 納品書・検収書
- ☑ 請求書（インボイス登録番号記載のもの）
- ☑ 銀行振込の通帳コピーまたはネットバンキング明細

**【広報費・ウェブサイト関連費】**
- ☑ 制作会社との契約書
- ☑ 成果物の確認書（画面キャプチャ等）
- ☑ 請求書・領収書
- ☑ 振込明細

**【外注費・委託費】**
- ☑ 業務委託契約書
- ☑ 業務完了報告書または成果物
- ☑ 請求書
- ☑ 振込明細

> **注意**: クレジットカード払いの場合、カード明細だけでは不十分なケースがあります。カード会社発行の利用明細と領収書を両方揃えておくと安全です。

## STEP4｜実績報告：「採択後の最大の難関」を乗り越える

事業が完了したら、**実績報告書**を所定の期日までに提出します。実績報告は補助金受け取りに直結する最重要ステップです。

不動産買取の現場では補助金とは縁が薄いように思われますが、業種を問わず、実績報告の書類不備で補助金額が減額された事業者を多く見てきました。ここでの失敗は取り返しがつかないケースもあります。

### 実績報告書の主な構成要素

1. **事業完了報告書**（実施した内容・成果）
2. **経費精算書**（支出一覧と補助対象経費の集計）
3. **支出証憑書類一式**（上記チェックリスト参照）
4. **成果報告書**（KPI達成状況、写真など）
5. **通帳コピー**（全ての補助対象経費の支払いが確認できるもの）

### 実績報告でよくある不備とその対策

| よくある不備 | 具体的な問題 | 対策 |
|---|---|---|
| 振込先の名義が違う | 代表者個人口座から払っている | 必ず事業名義・法人名義の口座を使用 |
| 証憑の日付が期間外 | 補助事業期間外の領収書 | 交付決定日〜完了期限内の日付か確認 |
| 相見積もりが不足 | 1社からの見積もりしかない | 基準額を超える場合は必ず複数社から取得 |
| 成果物の確認が不十分 | ウェブサイト制作の納品が不明確 | URLスクリーンショット・公開日の記録を取得 |
| 消費税の計上ミス | 免税事業者なのに税込で計上 | 課税・免税区分を確認して計上 |
| 経費区分の誤り | 対象外経費を含めて申請 | 交付決定通知書の経費明細と照合 |

## STEP5｜確定検査：審査担当者が実際に確認すること

実績報告書を提出すると、事務局による「**確定検査（かくていけんさ）**」が行われます。補助金によっては「完了検査」「書面審査」とも呼ばれます。

確定検査では、提出された書類と事業の実施内容が一致しているかを厳密に確認されます。場合によっては**現地確認（実地調査）**が行われることもあります。

### 確定検査で指摘されやすい3つのポイント

**① 経費の「実費払い」が確認できない**

補助金は原則として「実際に支払った経費」に対して交付されます。見積もり段階の金額と実際の請求書の金額が異なる場合は、差額の説明が必要です。値引きが発生した場合は**実際の支払額が補助対象**になります。

**② 事業目的と経費の関連性が不明確**

「この設備は採択された事業に使っているのか？」という観点で確認されます。たとえばパソコンを購入した場合、補助事業専用なのか社内の別業務にも使っているのか、使用割合の説明が求められることがあります。

**③ 補助事業期間内の支払いが証明できない**

補助金の補助事業期間は明確に定められています。**期間内に発注・納品・支払いが完了していること**が必要です。「納品は期間内だが支払いは翌月」というケースは要注意です。

## 体験談：実績報告で減額になりかけたAさんのケース

> Aさん（48歳・製造業・愛知県在住）は、ものづくり補助金で**加工機械の導入（補助申請額 **約380万円**）**に採択されました。しかし実績報告を自社で作成した際、機械の据付費用（**約45万円**）を経費明細に含めてしまいました。この据付費は交付申請時の経費明細に記載がなく、かつ「機械装置費」ではなく「その他経費」として計上していたため、事務局から不備の連絡が入りました。
>
> 幸い、据付作業が補助事業に必要なものであることを追加説明資料で証明できたため、最終的には補助対象として認められました。ただし修正書類の作成に**約2週間**を要し、入金が大幅に遅れることになりました。「交付申請の段階から、どの経費をどの区分で計上するか明確にしておくべきだった」とAさんは振り返っています。

このケースは、[補助金申請書の書き方｜審査員に選ばれる事業計画のコツ](/articles/2026-04-16-02)でも触れているとおり、交付申請段階での経費計画の精度が後工程に大きく影響することを示しています。

## STEP6｜補助金額の確定から入金まで：いつ、いくら振り込まれるのか

確定検査が完了すると、「**補助金確定通知書**」が届きます。この通知書に記載された金額が実際に振り込まれる補助金の確定額です。

申請した金額と確定額が異なるケースは珍しくありません。主な減額要因は以下のとおりです。

- 補助対象外経費が含まれていた
- 実際の支払額が見積もりより少なかった（値引きなど）
- 消費税の取り扱いの誤り
- 経費区分のミス

確定通知書を受け取ったら、まず**採択申請額・交付申請額・確定額の三者を比較**してください。大きな乖離がある場合は事務局に確認することをお勧めします。

### 入金（補助金振込）までのスケジュール感

確定通知書の発行から実際に入金されるまでの期間は、補助金の種類や事務局の混雑状況によって異なりますが、目安は以下のとおりです。

| フェーズ | 期間の目安 |
|---|---|
| 実績報告提出 → 確定検査完了 | **2〜8週間**（書類が揃っている場合） |
| 確定検査完了 → 確定通知書発行 | **1〜2週間** |
| 確定通知書発行 → 補助金振込 | **2〜4週間** |
| **合計（実績報告から入金まで）** | **1〜3ヶ月**程度 |

採択件数が多い回（特にものづくり補助金や事業再構築補助金の人気公募回）は、確定検査に時間がかかる傾向があります。「実績報告を出せば翌月に入金される」という期待は持たないほうが安全です。

> Bさん（55歳・小売業・大阪府在住）は、小規模事業者持続化補助金でウェブサイト制作に採択（補助金額 **約50万円**）されました。実績報告を期限ギリギリに提出したところ、確定検査で「ウェブサイトの公開日が補助事業期間最終日から2日後になっている」との指摘を受けました。

> 「実績報告は期限の2〜3週間前には出す心構えが必要だと痛感した」とBさんは語っています。

## 採択後に絶対に守るべき5つのルール

補助金の受け取りを確実にするために、採択後から入金まで守るべきルールをまとめます。

### ① 交付決定前に発注・契約しない

繰り返しになりますが、これが最大のタブーです。「早く進めたい」という気持ちは理解できますが、交付決定通知書が届いた日付以前の経費はすべて対象外になります。

### ② 経費の証憑書類は「その都度」保管する

後から「あのとき払ったけど書類がない」という事態を防ぐため、支払いのたびに請求書・領収書・振込明細をセットで保管してください。ファイリング方法はデジタルでもアナログでも構いませんが、**経費区分別に整理する習慣**をつけるとスムーズです。

### ③ 事業変更は事前に「変更承認申請」を行う

採択内容から事業内容・経費を変更する場合、勝手に変えてはいけません。**事前に事務局へ変更承認申請**を行い、承認を得てから変更してください。軽微な変更（一定割合以内の金額変動など）は事後報告で可能な場合もありますが、基準は補助金によって異なります。

### ④ 実績報告は期限の2〜3週間前に提出する

期限ギリギリに提出すると、書類不備があった場合に修正する時間がなくなります。余裕を持ったスケジュールを組むことで、不備への対応時間を確保できます。

### ⑤ 補助金受取後も帳簿・書類は**5年間（一部は10年間）**保管する

補助金によっては、入金後も一定期間は事業実施の証拠書類の保管が義務付けられています。中間検査や抜き打ちの実地調査が入る場合もあるため、書類はすぐ捨てないようにしてください。

## 補助金申請から入金まで：全体スケジュールの目安

最後に、公募〜入金までの全体的なスケジュール感を整理します。これはものづくり補助金を例とした目安です。

| フェーズ | 期間の目安 | 主な作業 |
|---|---|---|
| 公募開始〜申請締切 | **1〜2ヶ月** | 事業計画作成、書類準備 |
| 審査期間 | **2〜3ヶ月** | 待機（追加資料提出の可能性あり） |
| 採択通知 → 交付申請 | **2〜4週間** | 交付申請書・見積書の準備 |
| 交付申請 → 交付決定 | **3〜6週間** | 待機 |
| 事業実施期間 | **6〜12ヶ月**程度 | 発注・契約・支払い・証憑保管 |
| 実績報告 → 確定検査 | **2〜8週間** | 報告書作成・書類整理 |
| 確定検査 → 入金 | **1〜2ヶ月** | 待機 |
| **合計（申請から入金まで）** | **約1年〜1年半** | — |

「補助金は申請が通ればすぐもらえる」というイメージを持っている方は多いですが、実際には**申請から入金まで1年以上かかることも珍しくありません**。この認識を持ったうえで、資金計画を立てることが大切です。

## まとめ：補助金は「受け取り完了」まで気を抜かない

補助金の交付申請から入金までの流れを振り返ります。

- **採択後は交付申請が最初のステップ**。交付決定前の発注は絶対NG
- **事業実施中は証憑書類をその都度・正確に保管**することが最重要
- **実績報告は余裕を持って提出**し、期限ギリギリの提出は避ける
- **確定検査では経費の実費払い・事業との関連性・期間内支払い**が確認される
- **入金まで実績報告から1〜3ヶ月程度かかる**ことを資金計画に織り込む
- **補助金受取後も書類は5年以上保管**する義務がある

補助金は採択されただけでは一円も入ってきません。「申請前の計画」と「採択後の管理」の両方をしっかり進めることで、初めて補助金を事業の力に変えることができます。

書類準備や実績報告に不安がある場合は、中小企業診断士や認定支援機関などの専門家に相談することも有効な選択肢です。
