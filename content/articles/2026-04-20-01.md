## 補助金をもらったあと、会計処理で迷っていませんか？

「無事に採択されて補助金を受け取ったけれど、どう仕訳すればいいのかわからない」——そんな声を、補助金活用の現場で数多く聞いてきました。補助金は受け取るまでの申請作業に注目が集まりがちですが、受け取ったあとの会計処理を誤ると、税務リスクや決算誤りに直結します。

この記事では、補助金の仕訳・勘定科目の選び方、そして節税効果の高い「圧縮記帳」の仕組みまで、具体的な数字と仕訳例を交えながら解説します。経理担当者の方はもちろん、初めて補助金を受け取った事業主の方にも理解できる内容を目指しました。

## 補助金の会計処理が複雑になる理由

補助金の会計処理が難しいと感じる背景には、主に**3つの特殊性**があります。

**① 入金時期と費用発生時期がズレる**
補助金は、対象経費を先に支出してから後で受け取るのが基本です。このため、「費用が先・収益が後」という通常の会計とは逆の流れが生じます。

**② 収益計上のタイミングに選択の余地がある**
一般的な売上とは異なり、補助金は「いつの収益にするか」を会計処理の方針によってある程度コントロールできます。ここで圧縮記帳という選択肢が登場します。

**③ 勘定科目の名称が会社によって異なる**
「雑収入」「補助金収入」「国庫補助金受贈益」など、企業の会計方針によって使われる科目名が異なるため、統一的なルールを理解しておく必要があります。

2026年時点では、事業再構築補助金やものづくり補助金など大型補助金を活用する中小企業が増えており、会計処理の重要性はますます高まっています。

## 補助金に関わる主な勘定科目一覧

まずは、補助金の会計処理で登場する主な勘定科目を整理しましょう。

| 勘定科目 | 種類 | 使うタイミング |
|---|---|---|
| 未収入金 | 資産 | 補助金の交付決定が出たが未入金の段階 |
| 雑収入 / 補助金収入 | 収益 | 補助金を収益として計上するとき |
| 国庫補助金受贈益 | 収益 | 国・地方自治体からの補助金収益（法人税法上の用語） |
| 固定資産圧縮損 | 費用 | 圧縮記帳（直接減額法）適用時に固定資産を減額するとき |
| 圧縮積立金 | 純資産（剰余金） | 圧縮記帳（積立金方式）適用時に積み立てるとき |
| 繰延収益 | 負債 | 補助金を前受けする場合の一時的な計上先 |

> **ポイント**: 「雑収入」と「補助金収入」はどちらを使っても問題ありませんが、補助金の金額が大きい場合は「補助金収入」として独立した科目にする方が、後の管理・報告がしやすくなります。

## 入金タイミング別の仕訳パターン

補助金の会計処理は、**「交付決定時」「実際の入金時」「精算・確定時」**の3つのステップで考えるとわかりやすくなります。以下では、ものづくり補助金で**補助額 200万円**を受け取るケースを例に解説します。

### ケース①：交付決定時に未収入金を計上する場合

交付決定通知が届いた時点で、収益として確定させる処理です。

```
【交付決定時】
（借）未収入金  2,000,000  ／  （貸）補助金収入  2,000,000

【入金時】
（借）普通預金  2,000,000  ／  （貸）未収入金   2,000,000
```

この方法は、保守的な会計処理として広く使われています。交付決定が確定した時点で収益を認識するため、決算をまたぐ場合にとくに重要です。

### ケース②：入金時に一括計上する場合（小規模事業者に多い）

```
【入金時】
（借）普通預金  2,000,000  ／  （貸）補助金収入  2,000,000
```

処理がシンプルで、個人事業主や小規模な法人でよく採用されます。ただし、**交付決定が決算期をまたぐ場合は未収入金を使う方が適切**です。入金を待って計上すると収益の計上時期がズレ、税務上の問題になる可能性があります。

### ケース③：補助対象経費を先に支払うケース

補助金は通常、先に経費を支払ってから精算申請します。この流れも押さえておきましょう。

```
【設備購入時（例：機械装置500万円のうち補助200万円）】
（借）機械装置  5,000,000  ／  （貸）普通預金  5,000,000

【補助金入金時】
（借）普通預金  2,000,000  ／  （貸）補助金収入  2,000,000
```

この段階では、圧縮記帳を適用するかどうかを検討します。

## 圧縮記帳とは何か？節税効果のしくみ

圧縮記帳とは、補助金で取得した固定資産の**帳簿価額を圧縮（引き下げ）することで、補助金収益に対する課税を将来に繰り延べる**会計・税務上の特例処理です。

根拠となる法律は**法人税法第42条〜第50条**（国庫補助金等で取得した固定資産の圧縮額の損金算入）です。個人事業主の場合は所得税法第42条が根拠となります。

### 圧縮記帳の効果を数字で確認する

補助金 **200万円**を受け取り、**税率30%**と仮定した場合を比較してみましょう。

| | 圧縮記帳なし | 圧縮記帳あり |
|---|---|---|
| 補助金収益 | **200万円** | **200万円** |
| 圧縮損（損金） | 0円 | **▲200万円** |
| 課税所得への影響 | **+200万円** | **0円** |
| 当期の税負担増加額 | **約60万円** | **0円** |
| 翌期以降の減価償却費 | 多い（節税効果あり） | 少ない（圧縮分だけ減る） |

**重要な注意点**：圧縮記帳は「非課税」ではなく「課税の繰り延べ」です。固定資産の帳簿価額が下がった分、将来の減価償却費も少なくなり、その分だけ課税所得が増えます。あくまでも「今期の税負担を将来に先送りする」効果です。

## 圧縮記帳の2つの方法：直接減額法vs積立金方式

圧縮記帳には**直接減額法**と**積立金方式**の2種類があります。どちらを選ぶかは、会社の状況や決算書の見栄えによって判断します。

### 直接減額法

固定資産の帳簿価額を直接減らす方法です。処理がシンプルで、中小企業に広く採用されています。

```
【直接減額法の仕訳（補助金200万円を受け取り、圧縮記帳を適用）】

（借）固定資産圧縮損  2,000,000  ／  （貸）機械装置  2,000,000
```

この結果、機械装置の帳簿価額は **500万円 → 300万円**に圧縮されます。以降の減価償却は **300万円**を基礎として計算します。

### 積立金方式

固定資産の帳簿価額は変えず、剰余金（純資産）の中に「圧縮積立金」を計上する方法です。固定資産を正しい取得価額で管理したい場合や、外部への財務報告を重視する場合に選ばれます。

```
【積立金方式の仕訳】

①補助金入金時
（借）普通預金  2,000,000  ／  （貸）補助金収入  2,000,000

②圧縮積立金の計上（税効果会計を適用する場合）
（借）繰越利益剰余金  1,400,000  ／  （貸）圧縮積立金  1,400,000
　※税率30%の場合、税引後相当額 = 200万円 × (1-0.30) = 140万円

③毎期の積立金取崩し（減価償却に対応して取り崩す）
（借）圧縮積立金  ×××  ／  （貸）繰越利益剰余金  ×××
```

| 比較項目 | 直接減額法 | 積立金方式 |
|---|---|---|
| 処理の複雑さ | 簡単 | やや複雑 |
| 固定資産の帳簿価額 | 圧縮後の金額 | 取得原価のまま |
| 決算書への影響 | 総資産が減少 | 純資産内で処理 |
| 採用が多い企業 | 中小企業全般 | 上場企業・大企業 |
| 税効果会計 | 通常不要 | 必要な場合あり |

## 実例シミュレーション：Aさんの補助金会計処理

> **Bさん（40代・愛知県在住・製造業を営む法人の経理担当）のケース**
>
> 「ものづくり補助金で機械装置を**750万円**で購入し、補助金**500万円**を受け取りました。採択通知が届いたのが3月（決算月）の直前で、入金は4月。最初は入金時に収益計上しようとしていましたが、税理士に相談したところ『決算をまたぐので交付決定時に未収入金で計上すべき』と指摘されました。さらに圧縮記帳を適用することで、当期の税負担を大幅に抑えることができました。処理の手順を知らなければ、**約150万円分の税金を前倒しで払っていた**ことになります」

このBさんのケースを仕訳で整理するとこうなります。

```
【3月（決算月）：交付決定時】
（借）未収入金   5,000,000  ／  （貸）補助金収入  5,000,000

【4月：入金時】
（借）普通預金   5,000,000  ／  （貸）未収入金   5,000,000

【圧縮記帳（直接減額法）の適用】
（借）固定資産圧縮損  5,000,000  ／  （貸）機械装置  5,000,000

→ 機械装置の帳簿価額：750万円 → 250万円
→ 以降の減価償却は250万円を基礎に計算
```

補助金の採択通知が届いたタイミングで専門家に相談することが、こうしたミスを防ぐ最短ルートです。

[補助金申請書の書き方｜審査員に選ばれる事業計画のコツ](/articles/2026-04-16-02)でも触れていますが、採択後の手続きも申請と同様に計画的に進めることが重要です。

## よくある会計処理のミス5選とその対策

補助金の会計処理の現場でよく見られる誤りをまとめました。当てはまるものがないか確認してください。

### ❌ ミス① 入金があるまで収益計上しない

前述のとおり、交付決定が決算期をまたぐ場合は、決算時点で未収入金と補助金収入を計上する必要があります。入金を待つと収益の計上時期がズレ、**税務調査で指摘されるリスク**があります。

### ❌ ミス② 補助金で購入した資産を全額費用にする

中小企業の少額減価償却資産の特例（30万円未満）を利用できると思い込み、補助金対象資産を全額費用計上してしまうケースがあります。補助金対象資産は**圧縮記帳との関係を整理**してから処理方針を決める必要があります。

### ❌ ミス③ 消費税の扱いを誤る

補助金の受取額そのものは消費税の課税対象外（不課税）です。しかし、**補助金で購入した設備・物品にかかった消費税は、仕入税額控除の対象**になります。ここを混同して控除を失念するケースが見られます。

### ❌ ミス④ 圧縮記帳の適用漏れ

圧縮記帳は**確定申告書に圧縮限度額計算書を添付**しなければ適用できません。期限後に「やっぱり適用したい」と言っても認められないため、設備取得の時点で税理士と方針を確認しておくことが不可欠です。

### ❌ ミス⑤ 返還義務のある補助金を収益計上してしまう

補助金には交付条件によって**返還義務が発生するケース**があります（事業計画の未達成など）。返還義務が明確になっている場合は収益ではなく**負債（預り金）**として計上し、返還義務がなくなった時点で収益に振り替えるのが適切です。

## 個人事業主（青色申告）の補助金処理はどう違う？

法人の場合と基本的な考え方は同じですが、使う勘定科目や処理方法に若干の違いがあります。

| 項目 | 法人 | 個人事業主（青色申告） |
|---|---|---|
| 収益計上科目 | 補助金収入／雑収入 | 雑収入（事業収入に含める） |
| 圧縮記帳の根拠法 | 法人税法第42条〜 | 所得税法第42条〜 |
| 積立金方式 | 利用可 | 利用不可（直接減額法のみ） |
| 確定申告書の添付書類 | 圧縮限度額計算書 | 明細書（青色申告決算書に記載） |
| 消費税の扱い | 同じ（補助金本体は不課税） | 同じ |

個人事業主が圧縮記帳を使う場合、**直接減額法しか選べない**点は重要です。また、圧縮記帳の適用を受けるには**青色申告であること**が条件の一つとなります（所得税法施行令第92条）。

## 補助金の会計処理Q&A

**Q. 複数年度にわたる補助金はどう処理すればいいですか？**

A. 事業再構築補助金など、事業期間が複数年度にわたる場合は、各年度の事業進捗に応じて収益を分割計上する方法（進捗基準）が適切なケースもあります。ただし、交付条件や精算時期を確認して税理士と相談することをおすすめします。

**Q. 補助金で購入した資産を早期に売却・廃棄した場合は？**

A. 取得後一定期間（通常**5年以内**）に処分すると、補助金の一部または全部の返還を求められる場合があります。返還額が確定した時点で、収益の取り消し（借方：補助金収入または雑損失）として処理します。

**Q. 補助金の振り込みが分割になる場合の仕訳は？**

A. 交付額が確定している場合は、全額を未収入金で計上し、入金のたびに未収入金を減らす方法が一般的です。交付額が変動する可能性がある場合は、確定した金額ずつ計上していきます。

**Q. 採択されたが事業を途中でやめた場合は？**

A. 採択取り消しや事業中止によって補助金を返還する場合、すでに計上した補助金収益を取り消し（または返還損失として処理）します。固定資産の圧縮記帳を行っていた場合は、帳簿価額の再計算も必要になります。

## まとめ：補助金会計処理のチェックリスト

補助金の会計処理で押さえておくべきポイントを整理します。

### ✅ 仕訳・勘定科目のチェック
- [ ] 交付決定が決算期をまたぐ場合、決算時点で「未収入金／補助金収入」を計上しているか
- [ ] 入金時に「未収入金」を正しく消込しているか
- [ ] 補助金の受取額は「不課税」として消費税処理しているか
- [ ] 補助金対象経費にかかる消費税は仕入税額控除を適用しているか

### ✅ 圧縮記帳のチェック
- [ ] 圧縮記帳を適用するか否か、税理士と事前に方針を決めているか
- [ ] 直接減額法か積立金方式か、自社の状況に合った方法を選んでいるか
- [ ] 確定申告書に圧縮限度額計算書を添付しているか
- [ ] 圧縮後の帳簿価額を基礎に減価償却計算を見直しているか

### ✅ リスク管理のチェック
- [ ] 返還義務のある条件が交付規程に含まれていないか確認しているか
- [ ] 固定資産の早期売却・廃棄時の返還ルールを把握しているか
- [ ] 複数年度補助金の場合、各年度の収益計上方針を決めているか

---

補助金の会計処理は、受け取ったあとの処理を誤ると税務リスクや返還リスクに直結します。申請・採択の段階から会計処理の方針を決め、税理士と連携しながら進めることが、補助金を最大限に活用するための重要なステップです。

多くの事業者の現場を見てきた経験から言えることは、「採択されてから考える」では手遅れになるケースが少なくないということです。採択通知を受け取った時点で、すぐに会計処理の準備に取り掛かることを強くおすすめします。
